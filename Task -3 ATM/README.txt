# ATM Program

## Overview
This Java program simulates an ATM machine that interacts with a bank account. Users can check their account balance, withdraw funds, deposit money, and perform other banking operations.

## Instructions
1. The program prompts the user to select an operation from a menu.
2. Enter the number corresponding to the desired action.
3. For withdrawal or deposit, provide the amount as requested.
4. Follow the on-screen instructions for each operation.

## Features
- **Check Balance**: Displays the current account balance.
- **Withdraw**: Allows the user to withdraw a specified amount from the account.
- **Deposit**: Enables the user to deposit funds into the account.
- **Exit**: Quits the ATM program.

## Usage
- Upon starting the program, enter the desired operation by selecting the corresponding number.
- Follow the prompts to complete the chosen action.
- After completing an operation, the user can select another action or exit the program.

## Requirements
- Java installed on the system.
- Console input/output for interaction.

## Implementation Details
- The program is written in Java.
- It uses classes for bank accounts and the ATM functionality.
